================

# Core Staff Project
--------------------

This is the Core Staff project built with React, TypeScript, and Vite.

## Getting Started
---------------

To run the project, you need to set up the development environment. This project uses pnpm as the package manager.

### Install pnpm

If you haven't installed pnpm, you can install it by running the following command:

```bash
npm install -g pnpm
```

or

```bash
yarn global add pnpm
```

### Install Project Dependencies

After installing pnpm, you can install the project dependencies by running the following command:

```bash
pnpm install
```

This command will install all the required packages for the project.

## Run the Project
-----------------

After installing the project dependencies, you can run the project by running the following command:

```bash
pnpm dev
```

This command will start the project on port 3000, and you can access it by opening a web browser and navigating to:

```
http://localhost:5173
```

## Verify the Project
----------------------

After running the project, you need to verify that it's working correctly. You can do this by:

* Checking the user interface
* Testing the project's functionality
* Checking for errors

If you encounter any errors, you can check the log file to identify the cause.

## Other Commands
-----------------

* `pnpm build`: Builds the project
* `pnpm lint`: Checks the project for errors
* `pnpm test`: Runs the project's tests
